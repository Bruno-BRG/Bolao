# Documento de Desenvolvimento - Bolão da Copa

Este pacote contém:

- `documento_bolao_copa.pdf`: PDF final compilado.
- `documento_bolao_copa.tex`: fonte LaTeX.
- `diagramas/*.png`: diagramas renderizados usados no PDF.
- `mermaid/*.mmd`: fontes Mermaid equivalentes/editáveis dos diagramas.

Observação: o PDF inclui os diagramas já renderizados em PNG. Os arquivos `.mmd` foram incluídos para edição e futura recompilação dos diagramas em Mermaid.
